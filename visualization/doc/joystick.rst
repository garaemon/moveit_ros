Joystick Control of Rviz Interactive Markers
@@@@@@@@@@@@@@@@@@@@@@@@@@@@

Run
==================
roslaunch moveit_ros_visualization joy_sample.launch 

Joystick Cheat Sheet
==================

left analog stick: +-x/y
L2/R2: +-z
L1/R1: +-yaw
left/right: +-roll
up/down: +-pitch
triangle/cross: change planning group
square: plan
circle: execute
